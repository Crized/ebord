資料庫參數
名稱：ebord
編碼：utf8_general_ci

佈告資料
資料表名稱：table_Bulletin
欄位：(ID, subject, content, upload_Date, upload_Time)

使用方法
初始化資料庫 -> 進入公布欄操作

初始化資料庫

點選"公布欄操作" -> 點選"初始化" -> 填入資料庫資訊 -> 初始化完成

公布欄操作

1.點選"新增公告"
新增公告操作：1.編輯公告內容2.點選公布按鈕3. 公布完成
2.修改公告操作:
點選"公告清單" -> 點選"編輯按鈕(鉛筆圖示)" -> 編輯公告內容 -> 點選公布按鈕 -> 修正完成
刪除公告操作:
點選"公告清單" -> 點選"編輯按鈕(垃圾桶圖示)" -> 刪除完成
